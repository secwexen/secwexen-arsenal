# Project Roadmap

This roadmap outlines the planned development phases for the Secwexen Toolkit.

---

## Phase 1 — Foundation (Completed)
- Project structure created
- Initial tools added (offensive, defensive, OSINT, automation)
- Basic documentation
- CI/CD pipelines configured

---

## Phase 2 — Tool Enhancements (In Progress)
- Improve port scanner (multi-threading, service detection)
- Add real OSINT integrations (GitHub, IP APIs)
- Expand defensive tools with log analysis features
- Add more automation scripts

---

## Long-Term Vision
A fully modular, cross-platform security toolkit used for:
- Ethical hacking  
- OSINT investigations  
- System monitoring  
- Automation workflows
